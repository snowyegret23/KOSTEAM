# 개인정보처리방침

최종 수정일: 2026년 2월 1일

## 개요

KOSTEAM(이하 "확장 프로그램")은 사용자의 개인정보를 수집, 저장, 전송하지 않습니다.

## 수집하는 정보

이 확장 프로그램은 개인정보를 수집하지 않습니다.

## 로컬 저장 데이터

확장 프로그램은 브라우저의 storage API를 사용하여 다음 데이터를 사용자의 기기에만 저장합니다:

- 사용자 설정
  - 표시할 패치 소스 선택
  - 패치 정보 표시 설정
  - 검색 언어필터 우회 설정
  - 장바구니 기능 활성화 설정
- 캐시된 한국어 패치 정보 데이터
- 마지막 데이터 업데이트 확인 시간
- 장바구니 스냅샷 (복원 기능용, 임시 저장)

이 데이터는 외부 서버로 전송되지 않으며, 사용자의 브라우저 내에서만 사용됩니다.

## 외부 통신

확장 프로그램은 다음 목적으로만 외부와 통신합니다:

### GitHub (raw.githubusercontent.com)

- 한국어 패치 정보 데이터(JSON) 다운로드

### Steam (store.steampowered.com, api.steampowered.com)

장바구니 기능 사용 시, 사용자가 직접 버튼을 클릭할 때만 다음 통신이 발생합니다:

- 장바구니 복원: 삭제된 항목을 장바구니에 다시 추가
- 찜 목록 추가: 장바구니 항목을 찜 목록에 추가
- 게임 패키지 정보 조회: 장바구니 복원 시 appId를 packageId로 변환하기 위해 필요

Steam API 통신 시 Steam 페이지의 세션 토큰이 사용되며, 별도로 저장되거나 외부로 전송되지 않습니다.

## 제3자 제공

이 확장 프로그램은 사용자의 정보를 제3자에게 제공하지 않습니다.
